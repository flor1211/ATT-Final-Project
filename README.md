### Procurement KPI Analysis (2022–2023)
Dataset Source:  
[Kaggle Procurement KPI Analysis Dataset](https://www.kaggle.com/datasets/shahriarkabir/procurement-kpi-analysis-dataset)

The dataset contains procurement transactions including:
- supplier information
- procurement costs
- lead times
- defect rates
- compliance metrics
- procurement performance indicators
